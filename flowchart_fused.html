Build Fused Feature Cube

normalize (arr − min) / (max − min + ε) → stack along last axis.

② Pre-computation — Both Segmentation Modes
↻ FOR EACH MODE: [Temporal SLIC (use_flow=True), Standard SLIC (use_flow=False)]
Instantiate TemporalSLIC_DEM
Params: n_segments=1500, compactness=10.0, λ_z=0.5.
State: prev_labels = prev_centroids = prev_gray = None.
DEM stored as dem_norm (for SLIC channel) and dem_weighted = dem_norm × λ_z (for gradient).
↻ FOR EACH FRAME t = 0 … T−1  (calls tslic.segment(fused[t], use_flow))
Frame-Level Preparation
• Ensure input is (H, W, C) (expand dim if 2-D single channel).
• VIL proxy: curr_gray = (fused[:,:,0] × 255).astype(uint8) for Farneback.
• Append DEM channel: feature_cube (H,W,C+1) = concat[fused, dem_norm×λ_z].
◈ BRANCH — Is this Frame 0?
prev_labels is None
YES (Frame 0) NO (Frame t+)
Init Multi-Channel SLIC
Input: feature_cube (H, W, C+1) — satellite channels + DEM.
skimage.slic(feature_cube, n_segments=1500, compactness=10,
  enforce_connectivity=True, channel_axis=-1)
Clusters in (C+1)-D spectral + 2-D spatial space.
Output: labels (H, W) int32, starting at 1.
Init Compute Initial Centroids
scipy.ndimage.center_of_mass per unique label.
Store: prev_centroids = {lbl: [y, x]}.
(No flow computed — returns zero flow array.)
Step 1 Dense Optical Flow
if use_flow (Temporal mode):
cv2.calcOpticalFlowFarneback(prev_gray, curr_gray, ...)
pyramid: 3 levels, win=15, poly_n=5, σ=1.2.
Output: flow (H, W, 2) pixel displacements.
else (Standard mode): flow stays zeros.
Step 2 Advect Centroids (Lagrangian)
For each (lbl, [y_old, x_old]):
• Average flow over 10×10 pixel window centred at (y_old, x_old).
• y_new = clip(y_old + mean_flow_y, 0, H-1).
• Place marker: advected_markers[y_new, x_new] = lbl.
Step 3 Fused Gradient
Per channel c: sobel(fused_curr[:,:,c]).
Also: sobel(dem_weighted).
combined_gradient = pixel-wise MAX across all layers.
Boundaries snap to the sharpest edge in ANY spectral band.
Step 4 Compact Watershed
skimage.watershed(combined_gradient, markers=advected_markers,
  compactness=0.10).
Each advected marker grows to nearest gradient ridge.
Output: labels (H, W) int32.
Step 5 Recompute Centroids
self.prev_centroids = _calculate_centroids(labels).
Centers realigned to true region centroids of refined labels.
🔄 Update Lagrangian State
prev_labels ← labels  ·  prev_gray ← curr_gray.
Append labels, flow, prev_centroids to per-mode lists.
Continue to next frame.
← end frame loop → end mode loop →
Key simplification vs. Adaptive Residual version:  No warp-and-residual step. No Birth/Death thresholding. No RAG construction. Centroids are advected but never spawned or pruned — superpixel count is stable across frames.
