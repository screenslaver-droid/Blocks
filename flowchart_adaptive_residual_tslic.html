Frame 0 — Initialization
🔵 Multi-Channel SLIC
Append DEM as extra channel → (H,W,C+1) feature cube.
skimage.slic(feature_cube, n_segments=1500, compactness=10, enforce_connectivity=True)
Outputs label map (H,W) int32.
📍 Compute Initial Centers
center_of_mass per unique label.
Store prev_centers (K,2), prev_labels, prev_fused, prev_gray.
🕸 Build RAG
Scan horizontal + vertical pixel adjacencies.
Node features: mean fused vector per superpixel.
Edge weight: Euclidean feature distance.
Frame t+ — Adaptive Update
A Dense Optical Flow
if use_flow: cv2.calcOpticalFlowFarneback on VIL uint8 proxy.
Output: flow (H,W,2) — pixel displacement field.
A Advect Centers
For each superpixel k: compute mean flow over all its member pixels.
y_new = y_old + mean_fy, clipped to [0, H-1].
Key distinction: member-mean flow vs. centroid-point flow.
B Warp & Residual
Inverse-warp prev_fused using flow → pred_fused (H,W,C).
residual = max|fused_curr − pred_fused| across channels.
High residual ⇒ unexpected atmospheric change.
B ☠ Death Filter
For each advected center (cy, cx):
if max(fused_curr[cy,cx,:]) < thresh_death (0.05) → remove.
Kills superpixels in dissipated / empty space.
B 🌱 Birth Seeding
Threshold residual map → binary mask (>0.15).
Label connected components (blobs).
For each blob centroid: seed new center only if distance to all existing centers ≥ MIN_SPACING (20px) and blob intensity ≥ thresh_death.
C Watershed Refinement
Place integer markers at each adaptive center.
Gradient = pixel-wise max of per-channel Sobel + DEM Sobel.
skimage.watershed(combined_grad, markers, compactness=0.1).
Vectorised "Constrained SLIC" — no pixel-loop K-means needed.
C Recompute True Centroids
Update prev_centers from refined label map via center_of_mass.
D Build RAG
Same as Frame 0 procedure.
Scan adjacencies → compute edge feature-distances.
Returns {nodes, edges}.
🔄 Update Lagrangian State
Store: prev_labels ← labels, prev_fused ← fused_curr, prev_gray ← curr_gray.
Append labels, flow, rag to per-mode lists. Continue to next frame.
← end of frame loop → end of mode loop →
